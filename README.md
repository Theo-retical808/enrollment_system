# 🎓 Student Enrollment System

A comprehensive Laravel-based enrollment platform supporting both regular and irregular student workflows.

## 🚀 Quick Start

1. **Start XAMPP** (Apache + MySQL)
2. **Run:** `start-fast.bat`
3. **Access:** http://127.0.0.1:8000

## ✨ Features

- 🎯 **Dual Workflows:** Regular (auto-assignment) & Irregular (manual selection)
- 🔐 **Multi-Guard Auth:** Separate student and professor authentication
- 💳 **Payment System:** Enrollment fee verification
- 📋 **Smart Validation:** Prerequisites, conflicts, unit limits
- 🎨 **Blue Theme:** Professional, responsive UI
- ⚡ **Optimized:** Fast offline performance
