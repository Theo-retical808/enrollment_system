## Running the Seeder

To populate the database with this test data:

```bash
cd enroll_sys
php artisan db:seed --class=ComprehensiveCurriculumSeeder
```

Or to run all seeders including this one:

```bash
php artisan migrate:fresh --seed
```

## Test Scenarios Covered

1. **Regular Student Progression** - Students with no failures, following normal curriculum flow
2. **Irregular Student with Recent Failures** - Student currently retaking failed courses
3. **Irregular Student with Multiple Retakes** - Student who has failed multiple times and successfully retaken courses
4. **Prerequisite Blocking** - Students who cannot take advanced courses until prerequisites are passed
5. **Mixed Enrollment** - Irregular students taking both retakes and new courses simultaneously
